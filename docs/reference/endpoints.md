title: Endpoints
---

The Chromium driver comes with a set of many available endpoints.

All supported endpoints are inherited from the Appium base driver, and can be found in [their Appium docs reference pages](https://appium.io/docs/en/latest/reference/api/).
The driver does not define any additional endpoints. Refer to the documentation of your Appium
client for how to call specific endpoints.
